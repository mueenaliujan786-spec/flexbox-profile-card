# Profile Card (React Native)

This project is a simple Profile Card built using React Native and Flexbox.

## Features
- Profile Image
- Name and Role
- Two buttons (Follow & Message)
- Flexbox layout for button alignment

## Flexbox Used
flexDirection: 'row'
justifyContent: 'space-between'

## How it Works
The card is centered using Flexbox. Buttons are aligned in a row with spacing between them.
